Overview

Our interactive ML/DL platform is designed to bring advanced analytics, machine learning, and deep learning insights within reach for everyone, from students to seasoned data professionals. With intuitive tools and powerful models, this platform supports easy exploration, training, and testing across multiple data types and applications.

Dataset Exploration

Access preloaded datasets or add custom ones, exploring trends and patterns through interactive charts and visuals.

Model Training & Testing

Experiment with classification, regression, time series, and image classification models. Train new models or test existing ones with customizable parameters, displaying results and metrics.

Intelligent Chatbot

Powered by the ChatGroq API and enhanced with Hugging Face embeddings, our integrated chatbot assists users in navigating the platform, answering queries, and explaining model insights.

Data Visualization

Leverage a dedicated dashboard to understand model performance metrics, feature importance, and comparative insights across models.

Mission

This platform aims to bridge the gap between data and decision-making, fostering an accessible environment for learning and experimentation with real-world machine learning tools. By blending Flask and Streamlit, the interface is designed to streamline model evaluation, data analysis, and AI-driven insights for any user.

Flask

Manages page navigation, backend operations, and chatbot integration.

Streamlit

Facilitates interactive model testing, visualization, and dataset insights with chatbot integration.

ChatGroq API

Delivers an intelligent, conversational assistant, providing real-time support and guidance on data and model outputs.

Pandas

Used for data manipulation and analysis, enabling efficient handling of datasets and preprocessing tasks.
